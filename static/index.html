🤖 How to Use URDF Exporter

Export your Onshape CAD assembly to a URDF robot model in a few steps.

1
Get your Onshape API Keys

Go to cad.onshape.com → Account → API Keys, click Create New API Key, and copy the Access Key and Secret Key.

2
Enter API Keys in the sidebar

Open API Credentials in the left panel. Paste your Access Key and Secret Key. Check Remember on this device to avoid re-entering them next time — keys are stored locally in your browser only.

3
Paste your Onshape Document URL

Open your assembly in Onshape and copy the full URL from the browser address bar. It should look like:
https://cad.onshape.com/documents/abc123.../…
Paste it into the Document URL field.

4
Set the Assembly Name

Enter the exact name of your top-level assembly in Onshape (case-sensitive). Default is URDF_Top_Assembly. You can find the assembly name in the Onshape tab at the bottom of the editor.

5
Click Export and wait

Click the Export URDF button. The progress bar will track each stage (fetching parts, generating meshes, writing URDF). This usually takes 1–5 minutes depending on assembly complexity.

6
Preview and Download

Once done, the 3D viewer loads automatically. Use Joint Controls on the right to pose the robot. Click Download ZIP in the toolbar to save the complete URDF package.

💡 API keys are never stored on the server — only in your browser's localStorage.
